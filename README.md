# 👥 HR Analysis Dashboard
> An interactive Tableau dashboard analyzing workforce data — covering hiring trends, performance scores, department details, and employee turnover.

---

## 📊 Dashboard Overview

This project explores HR data to uncover patterns in employee hiring, demographics, performance, and termination reasons — helping HR teams make data-driven decisions.

---

## 🛠️ Tools & Technologies

| Tool | Purpose |
| Tableau | Data visualization & dashboard |
| CSV | Data source |

---

## 📈 Key Visualizations

- 📅 **Count of Employees Per Hiring Date** — Line chart showing hiring trends from 2006 to 2018
- 💍 **Employees By Marital Status** — Bubble chart breakdown by relationship status
- 🏢 **Departments' Details** — Treemap showing headcount & max salary per department
- 🌐 **Performance Per Recruitment Source** — Bar chart comparing performance % across hiring channels
- ❌ **Top 10 Termination Reasons** — Horizontal bar chart ranking exit reasons
- ⚧️ **Employees Per Gender** — Pie chart showing gender distribution

---

## 🔍 Key Insights

- 📅 Hiring **peaked in 2011** with **83 employees**, then declined sharply year over year
- 💍 **44% of employees are Single**, followed by Married at 40%
- 🏢 **Production** is the largest department with **209 employees**
- 💰 **IT/IS** has the highest max salary at **$220,450**
- 🔍 **Employee Referral** and **Online Web Application** achieved **100% performance scores**
- ❌ Top termination reason: **"Another position"** with 20 cases
- ⚧️ Workforce is **57% Female** and **43% Male**
